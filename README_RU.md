## 💡 Обзор

**DataExtractor TUI** — это высокооптимизированный кроссплатформенный инструмент командной строки, созданный для бесшовного сбора текстовых данных из множества файлов и директорий. Изначально разработанный для подготовки объемных кодовых баз под контекстное окно больших языковых моделей (LLM), он предлагает потрясающий терминальный интерфейс с нативной поддержкой мыши, умным определением кодировок и интеллектуальной фильтрацией файлов.

## ✨ Ключевые особенности

- **Продвинутый терминальный UI**: Плавная отрисовка, всплывающие модальные окна и обработка raw-ввода создают ощущение полноценного GUI прямо в консоли.
- **Нативная поддержка мыши**: Полная интеграция мыши в Windows (клики, наведение, выбор) через WinAPI.
- **Ноль зависимостей**: Написан исключительно на стандартной библиотеке Python. Никаких `pip install` не требуется.
- **Умная фильтрация**: Автоматически игнорирует бинарные файлы и стандартные директории разработки (`node_modules`, `.git`, `venv`, `__pycache__` и т.д.).
- **Drag & Drop**: Удобное перетаскивание файлов и папок напрямую в окно терминала.
- **Автоопределение кодировки**: Интеллектуально распознает и корректно читает `UTF-8`, `UTF-8-SIG`, `UTF-16`, `CP1251` и `CP1252`.
- **Сохранение состояния**: Запоминает ранее исключенные файлы, пути сохранения и языковые настройки.
- **Мультиязычность (i18n)**: Встроенная поддержка **русского**, **английского** и **китайского** языков.

## 🚀 Установка

Поскольку DataExtractor TUI **не имеет внешних зависимостей**, установка сводится к простому скачиванию скрипта.

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/data-extractor-tui.git

# Перейдите в директорию проекта
cd data-extractor-tui

# Запустите приложение
python main.py
```

## 🕹️ Использование

1. **Запуск**: Выполните команду `python main.py`.
2. **Выбор действия**: Нажмите `1` для начала извлечения или `2` для перехода в Настройки.
3. **Ввод пути**: Введите путь к директории вручную или просто **перетащите (Drag & Drop)** папку/файлы в терминал.
4. **Выбор файлов**: 
   - Введите номер файла, чтобы включить/исключить его из сборки.
   - Введите `0`, чтобы подтвердить выбор и запустить процесс извлечения.
5. **Результат**: Объединенные данные будут сохранены по указанному пути (по умолчанию в папку `Загрузки`) в формате `extracted_data_YYYYMMDD_HHMMSS.txt`.

### Управление (Клавиатура и Мышь)

| Действие | Сочетание клавиш / Ввод |
| :--- | :--- |
| **Навигация по меню** | Стрелки `Вверх` / `Вниз` |
| **Выбор пункта** | `Enter` или **Левый клик мыши** |
| **Назад / Отмена** | `Esc` или `Ctrl+C` |
| **Переключение файла** | Ввод номера файла (например, `1`, `4`) |
| **Старт извлечения** | `0` |

## ⚙️ Конфигурация

Приложение автоматически сохраняет ваши предпочтения (язык, директорию для вывода и память исключенных файлов) в локальный JSON-файл:

- **Путь**: `~/.merge_files_memory.json`

Вы можете безопасно удалить этот файл в любой момент, чтобы сбросить приложение к заводским настройкам по умолчанию.
