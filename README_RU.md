Интерактивный терминальный инструмент, который объединяет файлы проекта в один документ — готовый для вставки в контекст любой языковой модели.

## Что такое FileCLI?

При работе с AI-ассистентами (Claude, GPT, Gemini и другими) часто нужно передать кодовую базу в качестве контекста. FileCLI решает эту задачу: укажите папку, выберите нужные файлы, и за несколько секунд получите чистый объединённый `.txt`.

```
python merge_files.py
```

**Возможности:**

- TUI-интерфейс с управлением с клавиатуры — без сторонних зависимостей
- Выбор и отключение отдельных файлов; FileCLI запоминает выбор для каждой директории
- Поддержка Drag & Drop папок и файлов прямо в терминал (Windows, macOS, Linux)
- Автоматически пропускает бинарные файлы, артефакты сборки и `node_modules`
- Интерфейс на русском, английском и китайском языках

## Установка

FileCLI не требует сторонних пакетов — только Python 3.8 или новее.

**Клонировать и запустить:**

```bash
git clone https://github.com/Datvex/FileCLI.git
cd FileCLI
python merge_files.py
```

**Или скачать скрипт напрямую:**

```bash
curl -O https://raw.githubusercontent.com/Datvex/FileCLI/main/merge_files.py
python merge_files.py
```

## Использование

1. Запустите скрипт.
2. Нажмите `1`, затем введите путь к папке или перетащите её в терминал.
3. Переключайте файлы, вводя их номера. Нажмите `0` для подтверждения.
4. Готовый `.txt` появится в папке Загрузки (или по пути из настроек).

Выходной файл содержит чёткие разделители между файлами, так что любая языковая модель легко разберёт структуру.

## Настройки

Нажмите `2` в главном меню:

- **Путь сохранения** — изменить, куда сохраняются объединённые файлы
- **Язык** — переключить между русским, английским и китайским

Настройки сохраняются автоматически между сессиями.
